## 소개

Ptbwa Insight Pixel 템플릿은 이벤트 데이터(페이지뷰, 클릭, 스크롤, 전환)를 Ptbwa Insight 플랫폼으로 간편하게 전송할 수 있도록 단순화되어 있습니다. 광고주 및 마케터가 최소한의 설정만으로 사용자 인터랙션을 추적하고 전환 성과를 측정할 수 있도록 도와줍니다.

## 주요 기능

- **종합적인 추적 지원**: 페이지뷰, 클릭, 스크롤 깊이 등 표준 이벤트를 지원합니다.
- **전환 추적**: 구매, 회원가입, 주문 등 전환 이벤트를 커스텀 값 및 메타데이터와 함께 손쉽게 전송할 수 있습니다.
- **Google Ads 연동**: URL 쿼리 파라미터에서 `gclid`를 자동으로 캡처하고, 조건에 따라 쿠키에 저장하여 인터랙션 추적에 활용합니다.
- **샌드박스 JavaScript**: GTM의 보안 샌드박스 환경을 기반으로 구축되어 있습니다.
- **자동 ID 생성**: 쿠키를 통해 고유한 이벤트 및 사용자 식별자를 자동으로 처리합니다.

## 파라미터

| 파라미터             | 타입   | 설명                                                                                        |
| -------------------- | ------ | ------------------------------------------------------------------------------------------- |
| **Pixel ID**         | 텍스트 | Addirect에서 발급한 고유 ID입니다. (필수)                                                   |
| **Event Type**       | 선택   | page_view, click, scroll, conversion 중 선택합니다.                                         |
| **Scroll Depth**     | 텍스트 | 스크롤 좌표 값입니다. (Event Type이 'scroll'일 때 표시됩니다.)                              |
| **Click ID**         | 텍스트 | 클릭된 요소의 ID입니다. (Event Type이 'click'일 때 표시됩니다.)                             |
| **Conversion Type**  | 선택   | order, purchase, signup, signin 중 선택합니다. (Event Type이 'conversion'일 때 표시됩니다.) |
| **Conversion Value** | 텍스트 | 전환의 금액/가치입니다. (purchase/order일 때 표시됩니다.)                                   |
| **Conversion Meta**  | 텍스트 | 문자열 형태의 전환 추가 메타데이터입니다.                                                   |

## 설치 방법

1. 이 저장소에서 `template.tpl` 파일을 다운로드합니다.
2. **Google Tag Manager**를 열고 해당 컨테이너로 이동합니다.
3. **템플릿** > **태그 템플릿**으로 이동한 후 **새로 만들기**를 클릭합니다.
4. 우측 상단의 점 세 개 메뉴를 클릭하고 **가져오기**를 선택합니다.
5. 다운로드한 `template.tpl` 파일을 선택합니다.
6. **저장**을 클릭합니다.

## 사용 방법

1. GTM에서 **새 태그**를 생성합니다.
2. 태그 유형으로 **Ptbwa Insight Pixel**을 선택합니다.
3. **Pixel ID**를 입력합니다.
4. 추적할 **Event Type**을 선택합니다.
5. 적절한 **트리거**를 설정합니다. (예: `page_view`에는 모든 페이지, 클릭 이벤트에는 특정 클릭 트리거)
6. 컨테이너를 **미리보기**하고 **게시**합니다.

> 📖 **GTM이나 태그 템플릿이 처음이신가요?**  
> 템플릿 가져오기, 이벤트별 태그 설정, 트리거 구성, 미리보기 확인까지 단계별로 설명한 [**상세 사용 가이드**](docs/usage.ko.md)를 참고하세요.
